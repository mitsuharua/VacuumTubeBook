# VacuumTubeBook

# ディレクトリ構成

各ディレクトリの中には以下のようなファイルが入っています．

* Ep-Ip_characteristic/: 真空管のEp-Ip特性のグラフ
* LTspice/: LTspiceのモデルファイルと回路ファイル
* Program/: 真空管で実測したEp，IpからLTspiceのモデルファイルを作成するプログラム
